Parceiros

Nós da Efforia Nanocomputadores acreditamos que trabalhando unido com parceiros, crescemos mais forte. E é por isso que dispomos de diversos em nosso catálogo, de forma a estruturar e ajudar nosso empreendimento, onde quer que seja.



Projeto Cauã

O Projeto Cauã trata-se de uma iniciativa empreendedora de incentivo da constituição de novos empreendimentos que façam uso de nanocomputadores para as mais diversas áreas. A Efforia é apoiadora desta iniciativa e o projeto, nosso parceiro de negócios para diversas atividades-chave de nosso empreendimento.

Fábrica de Ideias Coworking

Através do programa de mentoria para startups, o projeto embrionário do Efforia, antes chamado de Caokzu, surgiu na Fábrica de Ideias Coworking o nosso negócio. Graças a todo o apoio que é dado até hoje para a companhia, continuamos como fiéis clientes e damos todo o suporte sobre a Tecnologia da Informação com os nossos produtos e serviços para o espaço coletivo.

Raspberry Pi Foundation

Graças a fundação Raspberry Pi foi possível realizar boa parte dos mais diversos projetos da empresa, devido a fabricação das placas Raspberry Pi Modelo B, uma de nossas placa-base precursoras para o início do projeto do HUB, antes conhecido como Projeto Pandora. A partir desta fonte, foi possível a geração de diversas outras placas-base pelo mundo todo e pela difusão desta tecnologia, da qual é trazida a você hoje pelos nossos nanocomputadores.

Arch Linux ARM

A partir do Arch Linux ARM é que foi possível constituir nossas bases de fundação de nossa tecnologia exclusiva EOS. Como forma de contribuição a tudo o que a comunidade desta variante de GNU/Linux ofereceu para a nossa plataforma, a Efforia tem um compromisso sério com o código aberto e disponibiliza quando possível tecnologias que podem ser utilizadas em favor deste sistema operacional.

Microsoft BizSpark

A partir do programa estruturante de startups da Microsoft, temos uma disponibilidade de recursos de nuvem e software que são ofertados de forma gratuita por três anos. Além da própria Efforia como beneficiária, recomendamos o uso do serviço dos mesmos toda a vez que acreditamos que há uma oportunidade para nossos coworkers no espaço coletivo onde a Efforia Nanocomputadores encontra-se instalado atualmente.




Código Aberto

A Efforia acredita que o desenvolvimento do Software Livre como um todo é um processo de extrema importância para a nossa evolução tecnológica, dado que boa parte de nossos produtos e serviços usufruem disso. Por isso, a companhia retribui regularmente contribuições em Código Aberto, sejam no âmbito de projetos de terceiros, como componentes do Django, assim como alguns softwares de apoio a desenvolvimento para a plataforma do Raspberry Pi e outras placas-base.



Uma companhia que apoia o software livre.

Temos uma fundação basicamente em tecnologias livres, como o Python, uma linguagem de programação, o Qt, um framework para construção de aplicativos com e sem interface gráfica, o Linux, um núcleo de sistema operacional e o Arch, um sistema operacional baseado no GNU/Linux. Assim como temos estas fundações, respeitamos padrões de engenharia de software e de sistemas para todas as nossas soluções; o EOS, para exemplo, cumpre as especificações dispostas na LSB (Linux Standard Base), em praticamente toda sua totalidade.

Se tem interesse em seguir nossos projetos open source, basta acessar nosso perfil no GitHub.
