Aplicações

A tecnologia exclusiva EOS, desenvolvida para os nanocomputadores da Efforia, garante a versatilidade a partir de um sistema de troca de cartões. Os programas desenvolvidos para a plataforma ficam auto-contidos em um cartão de memória, denominado de EOS Memory Card (EMC), compostos de um sistema básico específico para cada nanocomputador, um sistema intermediário idêntico para todos os nanocomputadores, e por fim, o aplicativo propriamente dito.


Um nanocomputador, múltiplas possibilidades

Graças a este diferencial, nossos nanocomputadores possuem diversas funcionalidades, todas elas divididas em cartões EMCs. Com isso, podemos utilizar um cartão para que o dispositivo funcione como um computador pessoal, dispondo de teclado, mouse e monitor, outro cartão para mídia center, conectado a uma televisão e a um controle sem fio, entre muitas outras possibilidades. Você escolhe como seu aparelho irá operar.


Com baterias inclusas


O HUB, equipado com a tecnologia EOS, consegue dispor o nanocomputador mais versátil para você através do sistema de troca de cartões. Desde um reprodutor multimídia até um computador multiuso, o sistema mostra toda a sua robustez.



Experiência surpreendente


Surpreenda-se com a experiência audiovisual que o HUB com EOS Play te proporciona. Reproduza filmes em até Full HD a partir de qualquer mídia que imaginar: interna, externa, pendrive, smartphone, tablet, entre muitas outras opções, tudo sem fio.



Carregue para onde quiser


Cabendo na palma de sua mão, o HUB é um dispositivo muito conveniente para carregar para todos os lados. Contatabém com suporte a tecnologia USB e HDMI, você consegue ligar vários periféricos a ele, como controles de jogo, teclados, pendrives e televisões.



Diversão ilimitada


O HUB possui uma extensa compatibilidade com os mais diversos sistemas de entretenimento, do Super Nintendo ao Playstation, tudo em uma pequena embalagem, sem contar com a gama de jogos disponíveis pelo HUB Pro através da plataforma Android.



Para o infinito e além


HUBs com o cartão EOS Cloud podem usufruir de um serviço chamado Efforia Cloud Service. Este serviço permite que seu HUB possa ser acessado remotamente pela internet, como e onde quiser.



Para os criativos e curiosos


Com o cartão EOS Beta para HUB e HUB Pro, você será capaz de explorar muito mais o potencial de seu aparelho, podendo utilizá-lo para diversas outras funções como a criação de programas e aplicações educacionais.
