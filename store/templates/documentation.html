Documentação

Neste local, disponibilizamos todo o material essencial para que você possa desfrutar o máximo de nossos nanocomputadores. Para começar, veja nosso guia de início rápido.



O que vem na embalagem?

O HUB, este Guia, uma Fonte de alimentação, um Cabo HDMI, um Cartão SD, um Wi-Fi USB, e um Adesivo do HUB.

O que há dentro do HUB?

O que há de mais recente e eficiente para um computador está dentro deste pequeno dispositivo.

Ligar os motores.

Para ligar o HUB, deve-se conectar na seguinte ordem: o cabo HDMI, o cartão SD, as conexões de rede e a fonte 5V.

Calibrar Acessórios.

O HUB dispõ de portas USB, que são utilizadas para ligar pendrives, leitores, adaptadores e HDs externos.

Preparar para a Largada.

O HUB consegue operar sem televisão, assim como sem roteador sem fio. Para a última opção, use o Hotspot.

Dada a largada!

Se você conseguiu efetuar todos os passos com sucesso, já poderá ligar seus dispositivos no HUB.

Como usar o EOS?

Para navegar, um controle ou um teclado conectado ao HUB já basta. Mouse e trackball são recomendados.

Meu HUB vem preparado?

Pré-instalado com EOS, o que traz um conjunto de aplicativos, o HUB também é expansível por meio de cartões SD.

Possibilidades Infinitas

Para poder ligar os seus dispositivos, basta acessar nosso site oficial para descobrir aplicativos para isso.

Expanda as possibilidades

O Efforia disponibiliza diversos acessórios certificados para você a partir de nosso site.
