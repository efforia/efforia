Desenvolvedor

Uma plataforma cheia de oportunidades para o desenvolvedor

Com a tecnologia exclusiva EOS, o desenvolvedor possui diversas vantagens que facilitam o trabalho de criação de novos serviços e aplicativos. A Efforia traz aos seus nanocomputadores tudo o que há de mais moderno para a engenharia de software e hardware, disponibilizando um sistema operacional multiplataforma, versátil e cheio de opções para você pré-instaladas, como a containerização, a implantação, o acesso remoto, a execução multilinguagem e a compilação distribuída.



Docker

O Docker é um serviço que disponibiliza virtualização leve para todos os nanocomputadores da Efforia. Fácil e prático de utilizar, ele faz uso da tecnologia de containers do Linux, responsável por trazer isolamento de contexto para os seus ambientes virtuais. Serviço indispensável para desenvolver ambientes com requisitos complexos a partir de poucos comandos e scripts, e manter a performance praticamente a mesma devida à fina camada de virtualização. Exemplo de uso é na instalação e na configuração de bancos de dados e servidores web.



Fabric

Com o serviço de implantação Fabric, é possível automatizar toda a instalação de configuração de componentes de software remotamente para o seu nanocomputador. Bastando a escrita de um script com os comandos necessários, muito do processo de entrada no modo produção do aplicativo desenvolvido é facilitado. Integrando o serviço com um controle de versão como o Git temos uma integração contínua perfeita para todos os softwares que deseja desenvolver para nossa plataforma.



Remote

O Remote mostra-se essencial para o acesso e administração remota de seu nanocomputador. Dispondo de serviço remoto de linha de comando (SSH), de acesso a arquivos por compartilhamentos do Windows (Samba), e de acesso a tela remota (VNC - quando o EMC possui interface gráfica), o Remote torna-se indispensável para o administrador de sistemas e para o programador realizar suas tarefas de manutenção e instalação de componentes no equipamento com a tecnologia EOS.



Runtime

Com o Runtime, é possível realizar a execução de diversas linguagens de programação em seu nanocomputador. O EOS vem preparado para ser utilizado para programação em Python, Perl, Ruby, Java, Node, PHP, C/C++ e C#, e um bom acervo de bibliotecas auxiliares para o desenvolvimento de aplicativos Web e com interface gráfica, como, respectivamente, o Django e o Qt. É com este serviço que nossos nanocomputadores demonstram a maior versatilidade, ao permitir diversas tecnologias a serem rodadas graças à tecnologia EOS.



Build

Tem um projeto grande demais para compilar com apenas um nanocomputador? Não se preocupe, o EOS conta com um sistema de compilação distribuída, o Build. O serviço conta com diversos mecanismos já instalados e configurados que dão a possibilidade de distribuir o processo em diversos outros nanocomputadores e computadores com a ferramenta EOS SDK instalada.



EOS SDK

A Efforia Nanocomputadores, pensando em você, desenvolvedor, disponibilizou um kit de desenvolvimento de aplicativos para a plataforma EOS. Neste conjunto há bibliotecas que compõem uma pequena camada sobre os principais componentes-chave de nossa tecnologia. O kit é complementado com uma máquina virtual em formato OVF (Podendo ser aberto em VMware, VirtualBox, Parallels e outros) que contém todos os itens do EOS mais a integração com os nanocomputadores, essencial para o uso do sistema Build. O SDK pode ser encontrado em nossa página do GitHub, e a VM pelo link: http://1drv.ms/1HoR5hN
