Entretenimento

Nossos nanocomputadores possuem um potencial gráfico bastante avançado, o que dá espaço para a criação de games para a plataforma. Pensando em demonstrar as possibilidades disso, instalamos em todos nossos equipamentos diversos sistemas de emulação de videogames com jogos à título de demonstração*
(Necessário ter a cópia original do jogo para executar).

Interface de fácil acesso

Através do controle do teclado sem fio incluso no HUB Pro, ergonomicamente fabricado para funcionar como um gampepad, o Retro dispõe de uma interface de uso bastante prático, bastando selecionar o sistema de emulação que você deseja, e em seguida, o game que você quer jogar. Boa parte dos jogos são compatíveis com os emuladores.

A solução para criação de jogos

Tendo à disposição diversas tecnologias multi-plataforma para gráficos, áudio e física, a criação de jogos para a plataforma torna-se bastante fácil e prática para os estúdios de jogos. Graças à tecnologia EOS-D somada ao nosso kit de programação, é possível conectar o HUB Pro ao notebook e utilizá-lo para testes e desenvolvimento.

Instalação de jogos prática

Para a instalação de jogos novos no Retro, é muito fácil: basta acessar o ponto de rede pelo computador chamado RETRO e copiar as roms para a pasta respectiva. O software trata de procurar, catalogar e colocar a arte do jogo, bastando apenas selecioná-lo na interface para jogar. Para outros, é mais fácil ainda: basta instalar diretamente na Google Play Store.

Entretenimento sob demanda

Nossos nanocomputadores, através de ferramentas como o Retro e a plataforma Android oferecida através da tecnologia EOS-D, tornam-se uma boa solução para entretenimento digital. Temos um acervo de outras soluções disponíveis para implantação no seu ambiente em caso de demanda especial, Contate-nos para saber mais.

Baterias inclusas para diversão

A Interface Retro instalada no coração de nossos nanocomputadores dispõe, através de um conjunto de tecnologias, acesso para 10 sistemas de emulação distintos, como Playstation, Nintendo 64, Game Boy Advance, Game Boy Color, Game Boy, Super Nintendo, Nintendo 8-bit, Sega Mega Drive, Sega Game Gear e Sega Master System. É diversão sem fim.
